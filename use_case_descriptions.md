## Use case description : 대여 금액 및 횟수 통계 조회

| Actor Action                                          | System Response                                                                         |
| ----------------------------------------------------- | --------------------------------------------------------------------------------------- |
| 1. 관리자가 '대여 금액 및 횟수 통계' 메뉴를 선택한다. | 2. 시스템은 기간 선택 옵션(1주일, 1개월, 1년)을 제공한다.                               |
| 3. 관리자가 원하는 기간을 선택한다.                   | 4. 시스템은 선택된 기간에 해당하는 총 대여 금액 및 총 대여 횟수 통계를 화면에 출력한다. |

## Alternative Courses

- **선택한 기간 내 대여 기록이 없는 경우**
  → 시스템은 "선택한 기간의 대여 기록이 없습니다"라는 메시지를 출력한다.
- **통계 데이터 계산 중 오류가 발생한 경우**
  → 시스템은 "통계 데이터 계산 중 오류가 발생했습니다"라는 메시지를 출력하고, 관리자에게 재시도를 요청한다.
